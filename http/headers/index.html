Module http::headers
[-]  [+] [src]

Types and utilities for working with headers in HTTP requests and responses.

This HTTP system is Special in that it uses lots of strong typing for its header system. All known HTTP headers are type checked, rather than being dealt with as strings all the time. Only unknown headers are stored in a map in the traditional way.

HeaderValueByteIterator

An iterator over the bytes of a header value. This ensures one cannot read past the end of a header mistakenly and that linear white space is handled correctly so that nothing else needs to worry about it. Any linear whitespace (multiple spaces outside of a quoted-string) is compacted into a single SP.
